제목 YBM, 전문 코딩능력 평가시험 'COS PRO' 출시
내용 ■3월 18일(일) 첫 시행, Python, C, C++, Java 컴퓨터 프로그래밍 언어능력 평가
■올 1월 민간자격등록 완료, 본인 능력에 따라 1급·2급 중 선택 응시 가능
YBM이 전문 코딩능력 평가시험 'COS PRO(Professional Coding Specialist)'를 출시하고 오는 3월 18일(일) 첫 시행한다고 밝혔다.

COS PRO는 Python, C, C++, Java 등 컴퓨터 프로그래밍 언어를 활용해 응시자의 소프트웨어 기반 사고력, 창의력, 논리력을 측정하는 시험이다. 시험 난이도에 따라 1급과 2급으로 나눠지며, 두 급수 모두 총 10문항으로 구성되어 있으며 1,000점 만점 중 600점 이상 점수 획득 시 합격 되는 방식이다. 수험자 본인의 실력에 맞춰 1급과 2급 중 선택해 응시할 수 있으며, 문제 출제 및 평가는 YBM에서 진행한다.
